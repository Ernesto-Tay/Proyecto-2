## ¿Qué y cómo funciona el BAWIZ SYSTEM?
El Sistema Bawiz es una aplicación diseñada para la gestión integral del negocio Bawiz Kiosco, ubicado en el Centro Comercial Mont Blanc.

Su objetivo principal es facilitar el control y registro de las operaciones diarias del kiosco, permitiendo al usuario administrar de manera eficiente la información relacionada con ventas, productos, proveedores, clientes y colaboradores.
A través de una interfaz amigable y organizada, el sistema permite:
- Registrar y visualizar ventas realizadas.
- Administrar el inventario de productos, incluyendo sus precios de costo y venta, existencias y descripciones.
- Gestionar la información de clientes frecuentes, proveedores y colaboradores del negocio.
- Generar un control estructurado de las operaciones diarias mediante una base de datos local.
